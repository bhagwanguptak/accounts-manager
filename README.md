# AccuHire Monorepo

This repo is organized as an npm workspace with two Next.js apps and a shared package.

Quick commands:

Install dependencies:
```bash
npm install
```

Run both apps concurrently:
```bash
npm run dev
```

Run individual app:
```bash
npm run dev:candidate
npm run dev:recruiter
```

Build the shared package:
```bash
npm run build:shared
```
